2. 설치와 실행
   javachip.apk 파일을 사용자의 안드로이드 폰에 설치합니다.
   javachip을 실행시킵니다,



3. 주요기능

   ? 로그인
	사용자가 아이디와 패스워드를 이용하여 시스템에 로그인
   ? 회원가입 
      	사용자가 어플리케이션을 사용하기 위해 회원가입을 해야 함
   ? 학습하기  
	사용자가 자신만의 학습 진도를 가짐
   ? 강의 제공
	각 단원에 맞는 강의 제공
   ? 개인 노트
	사용자에게 자신만의 노트 제공
   ? Q&A 
	사용자들 간 질문 및 답변 게시
   ? 문제풀이
	강의 별로 문제 풀이 제공


4. 사용 방법
	1. 어플리케이션을 실행한다.
	2. 로그인 화면에서 JOIN 버튼을 누르고 개인 정보를 입력한 뒤 회원가입을 한다.
	3. 회원가입이 완료되면 입력한 ID와 PW로 로그인 한다
	4. 사용자가 원하는 기능을 사용한다.
		- 학습하기 : 각 단원에 맞는 개념, 동영상강의, 문제풀이 제공
			학습완료 후 페이지에 다시 돌아와야 학습 진도가 갱신됨
		- 질문하기 : 사용자들 간 커뮤니케이션으로 질문하고 답변하기가 가능
			질문 작성 후 REFRESH 버튼 클릭으로 갱신해야 한다
		- 메모하기 : 자신의 메모장을 작성, 수정, 삭제 가능
		- 컴파일하기 : 코드를 입력하고 컴파일 결과 값을 얻을 수 있음
		- 사이드메뉴의 로그아웃 버튼을 누르면 로그아웃이 되고 종료할 수 있음
